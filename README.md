# alexanderwells.dev

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

Live at [alexanderwells.dev](https://alexanderwells.dev)

## Stack

- **Framework** - Next.js
- **Language** - TypeScript
- **Styling** - Tailwind CSS
- **Animations** - Framer Motion
- **Hosting** - Vercel
- **AI Feature** - Ask Alex chatbot
